# Experiment 51: Division with Exception Handling

## Problem Statement

Write a Java program that reads two integers $a$ and $b$ and calculates $a / b$.
Use a `try-catch` block to handle `ArithmeticException` if $b$ is zero.
* If an exception occurs, print "Divide by zero error".
* Otherwise, print the result.

## Input Format

* Two integers $a$ and $b$ (space-separated or on separate lines).

## Output Format

* The result of division OR "Divide by zero error".

### Example 1

**Input:**

```text
10 2
```

**Output:**

```text
5
```

### Example 2

**Input:**

```text
10 0
```

**Output:**

```text
Divide by zero error
```
